# 24点游戏设计介绍
## 基本功能
输入扑克牌中四个数通过加减乘除计算出24，显示计算过程，并提供成功或失败信息。
### 代码内容
在这个程序中，我们使用了递归的方式来穷举所有可能的计算方法。程序首先接受用户输入的四个数字，然后调用 `solve` 函数来进行计算。`solve` 函数中使用了两层循环遍历所有数字的组合，然后对每一对数字进行加减乘除运算，并递归调用 `solve` 函数进行下一轮计算，直到剩下一个数字。最后，判断是否找到了一个有效的解，并打印出计算过程和结果。
请注意，由于浮点数的精度问题，我们在判断计算结果是否为24时，使用了一个允许的误差范围 `EPSILON`。如果需要精确计算结果，请使用高精度的浮点数库或整数计算
